Gas Stations Problem

Supply a file in which each line is a string/array consisting of the following elements: N which will be the number of gas stations in a circular route and each subsequent element will be the string g:c where g is the amount of gas in gallons at that gas station and c will be the amount of gallons of gas needed to get to the following gas station. For example one line may be: ["4","3:1","2:2","1:2","0:1"].

The goal is to return the index of the starting gas station that will allow you to travel around the whole route once, otherwise return the string impossible.
